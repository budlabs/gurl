`gurl` - simple plumbing framework

SYNOPSIS
--------
```text
gurl [--matchers-dir|-d DIR] URL [ARG...]
gurl --help|-h
gurl --version|-v
```

DESCRIPTION
-----------
LONG DESCRIPTION


OPTIONS
-------

`--matchers-dir`|`-d` URL  

`--help`|`-h`  
Show help and exit.

`--version`|`-v`  
Show version and exit.


ENVIRONMENT
-----------

`XDG_CONFIG_HOME`  

defaults to: $HOME/.config

`MATCHERS_DIR`  

defaults to: $HOME/.config/gurl

DEPENDENCIES
------------
`bash`
